Что надо сделать для запуска.
1. Скопировать содержимое папки yii на хост (на хосте должен быть шаблон yii basic)
2. Подгрузить в базу таблицы  - файл yii.sql
3. Установить настройки соединения с сервером базы данных и адрес отправителя электронной почты (файлы - yii/config/db.php(настройка параметров базы данных), yii/config/param.php(настройка адреса отправки электронной почты))
